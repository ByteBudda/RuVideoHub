## 🚀 Что это на самом деле

Построенная на **Jetpack Compose** с эффектом **Liquid Glass**, архитектура приложения спроектирована так, словно Apple, Google и VLC объединили усилия в 2026 году.

- **Мозг приложения:** `SmartRutubeParser` с 15+ эвристиками распознавания контента
- **Интеллектуальная защита:** 7 уровней фильтрации платного контента и подписок
- **Дизайн:** Liquid Glass UI, адаптирующийся под Mobile и TV (D-pad support)
- **Оффлайн:** Прямое скачивание, управление прогрессом и экспорт

---

## 🧠 SmartRutubeParser — Сердце RuVideoHub

Наш парсер не просто запрашивает API — он анализирует структуру объекта, взвешивает эвристики и нормализует данные на лету.

**Возможности парсера:**

- **Универсальный детектор** — распознаёт видео, сериалы, каналы и плейлисты по URL-паттернам
- **Адаптивный экстрактор** — игнорирует переименования полей API (aliases: picture_url, poster_url, image)
- **Защита от paywall** — обходит блокировки PREMIER, START, IVI, KION и других
- **Пагинация** — автоматическое извлечение next_url, has_next, page
- **Рекурсивный парсинг** — распаковывает вложенные структуры (cards, items, results)

## 🏗️ Архитектура

Приложение построено на принципах чистой архитектуры:

```
┌─────────────────────────────────────────────────────┐
│                   UI Layer (Compose)                │
│    Home  │  Explore  │  Library  │  Settings        │
├─────────────────────────────────────────────────────┤
│                  ViewModel Layer                    │
│        VideoViewModel — единый источник правды      │
│            StateFlow для всех экранов               │
├─────────────────────────────────────────────────────┤
│                  Repository Layer                   │
│      VideoRepository — кэш, парсинг, API            │
│      SmartRutubeParser — универсальный парсер       │
├─────────────────────────────────────────────────────┤
│                  Data Layer                         │
│    Room DB  │  DataStore  │  OkHttp  │  ExoPlayer   │
└─────────────────────────────────────────────────────┘
```

**Технический стек:**

| Компонент | Технология |
|-----------|------------|
| UI | Jetpack Compose, Material3 |
| Архитектура | MVVM, Clean Architecture |
| DI | Hilt (Dagger) |
| Плеер | Media3 (ExoPlayer) + HLS |
| Локальные данные | Room Database, DataStore |
| Асинхронность | Kotlin Coroutines & Flow |
| Сеть | OkHttp3, Retrofit |

## 🎬 Плеер: Профессиональный уровень

Три уровня отказоустойчивости (HLS → Retries → WebView Fallback) обеспечивают воспроизведение в 99.9% случаев.

**Жесты премиум-класса:**

| Жест | Действие |
|------|----------|
| Двойной тап слева | -10 секунд |
| Двойной тап справа | +10 секунд |
| Вертикальный свайп слева | Яркость экрана |
| Вертикальный свайп справа | Громкость |

**Соотношения сторон (VLC-style):**
- Вписать (FIT)
- Заполнить (FILL)
- Растянуть (STRETCH)
- 16:9
- 4:3
